NAME

x2gostartagent - X2go Server Script  

SYNOPSIS

x2gostartagent <geometry> <link-quality> <nx-pack-meth> <cache-type+dpi> <kblayout> <kbtype> <setkbd> <session-type> <command>

 

DESCRIPTION

x2gostartagent is a script that coordinates the startup process of new X2go sessions. It expects quite a list of parameters, all of which are obligatory.  

RETURN VALUES

A list of parameters will be returned after a successful session startup. Each value will be presented on a new line:


    <X-display-number>
    <session-cookie-hash>
    <x2goagent-pid>
    <session-name>
    <graphics-port>
    <sound-port>
    <sshfs-port>
